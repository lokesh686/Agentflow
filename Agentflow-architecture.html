Agentflow Architecture

A monorepo web application with a React frontend, Node.js API gateway, and a Python orchestrator.

Frontend

  • • React/Vite application
  • • Runs on port 5173
  • • Interacts with the API Gateway

Backend

  • • API Gateway (Node.js) on port 3000
  • • Orchestrator (Python) on port 8001
  • • Both connect to MongoDB and Redis

Infrastructure

  • • MongoDB for data storage
  • • Redis for caching
  • • Managed by Docker Compose
